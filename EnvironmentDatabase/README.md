# Environment Database
Database for storing environmental data and metrics near pollen sensir monitors. Note: might be merged with pollen database in the near future. 

**Summary** <br>
This GitHub repo contains python scripts, custom classes, and a PostGIS database design to curate a database, develop and implement a moderate resolution (e.g. neighborhood level) pollen model for core based statistical areas where SPIROMICS participants live.
